Bulk Insert from Excel File

Please follow the following instructions carefully:

  • Step 1: In an Excel File, create and rename the sheet to be the course code for each course that you would like to upload lab schedules for
  • Step 2: Copy-paste the class schedule table (including headers) directly from NTU Class Schedule Portal/STARS
  • Step 3: Ensure that the top row consists of the fields -- TYPE, GROUP, DAY, TIME, VENUE, REMARK
  • Step 4: Upload this Excel file in the field below, and press the 'Upload' button to import the schedules

Notes:

  • You may check and edit the successfully imported lab schedules in the Course information from Course Overview page.
  • This tool only handles the LAB class information.
  • The name of your file does not matter.
  • The sheet will be ignored if the course has not been added into the database at the time of this bulk-import.
  • The sheet name should correspond to the 1st course code in the database. (e.g SC1005/CE1105/CZ1105: name your sheet SC1005)

⚠️ Wrong formats will result in the system's inability to import the data.
